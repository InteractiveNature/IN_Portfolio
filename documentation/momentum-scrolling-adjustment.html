Work Gallery Momentum Scrolling Adjustment

This document explains how to adjust the momentum scrolling behavior of the work gallery on mobile devices.

Note: These adjustments primarily affect touch-based scrolling on mobile devices. Desktop scrolling behavior remains largely unchanged.

Understanding the Parameters

The momentum scrolling behavior is controlled by two main parameters:

Parameter Description Default Value
maxMomentumVelocity The maximum velocity (in pixels per frame) that the gallery can scroll, regardless of how quickly the user flicks. Lower values result in slower, more controlled scrolling. 4
momentumScaleFactor The factor by which the initial velocity is scaled. Higher values make the gallery more responsive to quick flicks, but can make it harder to control. 15

How to Adjust the Settings

The website includes a global helper function that allows you to easily adjust these parameters from the browser console. Here's how to use it:

  1. Open the website on your mobile device
  2. Open the browser's developer tools (this varies by browser, but usually involves tapping a menu option and selecting "Developer Tools" or similar)
  3. Navigate to the Console tab
  4. Use the adjustMomentumSettings function as shown below

Function Syntax

adjustMomentumSettings(maxVelocity, scaleFactor);

Both parameters are optional. If you omit a parameter, its current value will be retained.

Examples

// Set maximum velocity to 2 (slower, more controlled scrolling)
adjustMomentumSettings(2);

// Set scale factor to 10 (less responsive to quick flicks)
adjustMomentumSettings(undefined, 10);

// Set both parameters
adjustMomentumSettings(3, 12);

// Check current settings without changing them
adjustMomentumSettings();

Recommended Values to Try

Here are some recommended combinations to experiment with:

Use Case maxMomentumVelocity momentumScaleFactor Code to Run
Very slow, controlled scrolling 1.5 10 adjustMomentumSettings(1.5, 10);
Moderate scrolling (default) 4 15 adjustMomentumSettings(4, 15);
Responsive but controlled 3 12 adjustMomentumSettings(3, 12);
More responsive 6 15 adjustMomentumSettings(6, 15);

Making Permanent Changes

Once you've found settings that work well, you can make them permanent by updating the initialization in js/main.js:

const workItemManager = new WorkItemManager({
    selector: '.work-grid',
    edgeScrollEnabled: true,
    edgeScrollThreshold: 150,
    edgeScrollSpeed: 5,
    hoverInteractionEnabled: true,
    bounceAnimationEnabled: true,
    maxMomentumVelocity: 3, // Your preferred value
    momentumScaleFactor: 12 // Your preferred value
});
Tip: Test on multiple devices if possible, as touch sensitivity can vary between different phones and tablets.

Troubleshooting

If you encounter issues with the momentum scrolling:

  • Make sure you're testing on a touch-enabled device
  • Try refreshing the page after making changes
  • Check the console for any error messages
  • If the gallery becomes unresponsive, reset to defaults with adjustMomentumSettings(4, 15);

For further assistance, contact the development team.
